##### AVISO: O projeto foi implementado em distribuição LINUX. #####

## Para a execução do projeto é necessario ter o python3 instalado. Caso você não tenha o python,
instale através do seguinte código:

$ make setup

## Com o python instalado, execute o programa com o seguinte código:

$ make run
ou
$ python3 main.py

## Caso tenha alguma dúvida de como gerenciar os arquivos, digite, no FatShel:

$ help

## Para sair do FatShel digite:

$ quit
